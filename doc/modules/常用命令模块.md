# 常用命令模块（view-commands）

> 返回 [项目架构深度分析](../项目架构深度分析.md)

---

## 一、概述

展示 OpenCode CLI 和 TUI 命令参考，按分类卡片显示，支持视图级命令列表展示与输入框中的 `/` 命令面板。

**前端**：`commands-view.js`、`command-palette.js`

**后端**：`commands.go`（常用命令视图静态分组数据）、`service/tree.go` / `app.go`（命令面板动态命令）

---

## 五、实现

### 5.1 TAB 切换

```javascript
// commands-view.js
document.querySelectorAll('.cmd-tab').forEach(tab => {
    tab.addEventListener('click', () => {
        // 切换 active 类
        // 加载对应 tab 的命令列表
        loadCommands(tab.dataset.cmdTab);
    });
});
```

### 5.3 `/` 命令面板

`command-palette.js` 在输入框中键入 `/` 时加载命令面板：

- 固定命令：`summarize` / `revert` / `unrevert`
- 动态命令：`api.GetOpenCodeCommands()` → `app.go` → `service.GetOpenCodeCommands()` → `/command`
- 合并后支持键盘上下选择、Enter/Tab 选中和鼠标点击

### 5.4 复制到剪贴板

```javascript
card.addEventListener('click', () => {
    navigator.clipboard.writeText(command);
    showToast('已复制: ' + command, 'success');
});
```
