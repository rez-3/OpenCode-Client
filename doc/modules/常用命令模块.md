# 常用命令模块（view-commands）

> 返回 [项目架构深度分析](../项目架构深度分析.md)

---

## 一、概述

展示 OpenCode CLI 和 TUI 命令参考，按分类卡片显示，点击复制到剪贴板。

**前端**：`commands-view.js`、`commands.go`（后端静态数据）

---

## 二、页面结构

```
┌──────────────────────────────────────┐
│  [CLI 命令]  [TUI 命令]              │  ← TAB 切换

│  │  run  | 启动交互式对话       │    │

```

## 五、实现

### 5.1 TAB 切换

```javascript
// commands-view.js
document.querySelectorAll('.cmd-tab').forEach(tab => {
    tab.addEventListener('click', () => {
        // 切换 active 类
        // 加载对应 tab 的命令列表
        loadCommands(tab.dataset.cmdTab);
    });
});
```
